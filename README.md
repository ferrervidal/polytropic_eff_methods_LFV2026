# polytropic_eff_methods_GT2026
Reference implementation accompanying ASME paper on polytropic efficiency.

This repository contains reference Python code accompanying the paper:

Ferrer-Vidal España-Heredia, L.E., "REVISITING THE POLYTROPIC EFFICIENCY AND ITS PRACTICAL USE,"

## Purpose

The code demonstrates several commonly used methods for applying polytropic
efficiency in compressor performance calculations, including perfect-gas,
thermally perfect, and real-gas formulations.

The implementation is intended for instructional and comparative purposes.
It prioritizes clarity and traceability over computational efficiency or
generality.

## Contents

- `polyEffmethods_GT2026-177067.py`  
  Reference implementation of the polytropic efficiency methods discussed
  in the paper.

- `thermoWF_GT2026.py`  
  Minimal thermodynamic property module and state container used by the
  reference implementation.

## Disclaimer

This code is provided as a research and instructional reference only.
It is not a validated design or analysis tool.
